1. Ставим проприетарный драйвер для видеокарт:
	Заходим в «программы и обновления»
	во вкладку «дополнительные драйверы» 
	необходимо, чтобы был выбран пункт с NVIDIA driver metepackege. (По умолчанию выбран X.org X server) 

2. Копируем скачанные файлы в домашнюю папку. Обязательно в домашнюю папку, иначе работать не будет.

3. Проверяем, что файл GPUcontrol является исполняемым: 
	ПКМ — Свойства — Права — Должна стоять галочка после слова «Выполнение» 
	(Разрешить выполнение файла как программы). 

4. Чтобы команды через SUDO в скрипте выполнялись без запроса пароля:
	запускаем терминал и в нем пишем 
	
	sudo visudo
	
	нажимаем Enter и у вас попросит пароль от учетной записи, вводим его и ниже строк
	
	# Allow members of group sudo to execute any command
	%sudo   ALL=(ALL:ALL) ALL
	
	вводим такую строку
	
	m1 ALL=(ALL) NOPASSWD: ALL
		где  "m1" имя вашего пользователя
	
	чтобы сохранить изменения нажимаем  CTRL+O, Enter
	для выхода из редактора  CTRL+X

5. Добавляем скрипт в автоагрузку. 
	В убунту заходим в «настройки запуска приложений» 
	нажимаем кнопку добавить 
	Пишем имя, которое вам удобно (я пишу GPUcontrol)
	нажимаем кнопку обзор и указываем путь к файлу 
	описание можно не писать, и нажимаем добавить.

6. Открываем файл GPUsettings в текстовом редакторе и меняем значения как вам необходимо
	
7. Если надо изменить задержку запуска скрипта, то открываем файл GPUcontrol в текстовом редакторе 
и меняем значение параметра 
	sleep 30	задержка запуска скрипта в секундах	
   Сохраняем (CTRL+S)

8. Скрипт сам создаёт настроичный файл Xorg.conf и пытается перезагрузить Х сервер.
Если вдруг у вас не заработает с первого раза, то просто перезагрузите компьютер и всё должно заработать. 

9. Пользуемся с удовольствием!
